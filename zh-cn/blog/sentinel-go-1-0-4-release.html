Sentinel Go v1.0.4 发布

Sentienl Go v1.0.4 版本正式发布，带来了 Prometheus metric exporter、熔断恢复策略强化、Apollo 数据源等多项特性与改进。同时，我们也对 Sentinel Go Kubernetes CRD data-source 进行了升级更新，从而可以支持最新的 Kubernetes 1.22 及以上版本。

Release notes：https://github.com/alibaba/sentinel-golang/releases/tag/v1.0.4
